Visual Composer Extension that will allow user to add 150+ hover effects to images with animated captions.
== Description ==
Hover Effects Pack - Visual Composer Extension is collection of 150+ hover effects that will allow you to add animated captions on images when hover over the images.It contains 10+ captions animations and it's 100% responsive on all devices.

<a href="http://demo.webdevocean.com/hover-effects-pack-demo/">See DEMO Here</a><br>

== Installation ==
<b>Note:</b> <b><i>You require Visual Composer Installed to use this add-on.</i></b>

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Hover Effects Pack- Visual Composer Extension' and install it.
